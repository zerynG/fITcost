Страница управления проектом - страница редактирования (создания/просмотра - в зависимости от прав) создаваемого проекта, куда вносится вся необходимая информация.

Реквизиты проекта:
- Название;
- Интервал выполнения;
- Описание проекта;
- Заказчика (не обязательно, выбор из реестра Заказчиков);
- Техническое задание (не обязательно, переход в конструктор документа);
- Формирование расчета НМА (нематериального актива) -  Форма стоимости НМА;
- Формирование коммерческого предложения - горит если выбран Заказчик, - Форма Коммерческого предложения;
- Стоимость проекта (итоговая стоимость по всем статьям расходов + налоговая ставка). Считается по формуле: isp = sp + (sp*ns), где isp - итоговая стоимость проекта c учетом налога, sp - итоговая стоимость проекта с маржинальностью, ns - указанная налоговая ставка в %;
- Себестоимость проекта;
- Стоимость с маржинальностью (если выбран Заказчик и она стоит в услугах);
- Чистая прибыль (если стоит маржинальность) - сумма сумм разницы между итоговой стоимостью услуг и стоимостью маржинальности. Считается по формуле (пример): p = (i - s) + (i -s) .... - и так по каждой услуге. В данной формуле: p - чистая прибыль, i - итоговая стоимость услуги, s - себестоимость услуги;
- Налоговая ставка - % указание пользователем - влияет на финальный расчет;

Ниже в другой вкладке "Управление ресурсами проекта";
Управление ресурсами проекта - лист (список) всех задействованных кадров (сотрудников), подрядчиков и ресурсов, которые могут быть задействованы для реализации данного проекта.
ВАЖНО: Каждый ресурс добавляется из реестра (список ресурсов).

Вид отображения в списке:
- Название ресурса;
- Тип ресурса;
- Выбор исполнителя;
- Название услуги (которую оказывает ресурс);
- Интервал (выбор интервала времени + дата);
- Себестоимость;
- Маржинальность;
- Итоговая стоимость;


Каждый отдельно с расчетами:

**Название ресурса** 
Вписываемое в текстовое поле название, которое будет отображаться в документах (Коммерческое предложение и Стоимость нематериального актива).

 **Тип ресурса**
Тип ресурса влияет на реестр выбора, а также систему расчета в реестре:
- Сотрудник - реестр сотрудников;
- Исполнитель - реестр физических лиц;
- Субподрядчик - реестр юридических лиц;
- Оборудование - реестр оборудования;
О расчетах каждого описано на странице реестра

**Выбор исполнителя**
Выбор объекта из реестра с возможностью поиска. В данном поле должна быть фильтрация в зависимости от того, какой  тип ресурса был выбран выше.

**Себестоимость**
Себестоимость - стоимость использования ресурса в расчете стоимость за единицу измерения умноженную на количество часов или дней (в зависимости от расчета в реестре).

**Маржинальность**
Маржинальность - % который начисляется сверху на стоимость, который отображается только руководителю.

**Итоговая стоимость** - стоимость услуги с учетом маржинальности.
Считается по формуле: i = s*m
i - итоговая стоимость;
s - себестоимость;
m - % маржинальности;
